h1. Bandwitdh meter tests

h2. Environment

* One OM2P-HS and one OM2P-LC nodes using ath9k based wifi devices connected in ad-hoc mode on *2.4GHz*.

h3. Scenarios

# bw meter from HS to LC (and viceversa)
# iperf from HS to LC (and viceversa). Iperf was running on the nodes themselves
# HTTP download from an host behind the HS to another one behind the LC (and viceversa). In particular:
## host connected to the Ethernet port of the node
## Ethernet interface bridged with the batman-adv soft interface

Test in scenarios 1 and 2 were executed alternatively (one bw meter, one iperf, one bw meter, one iperf, ...) while the HTTP downloads have been executed all together later on.
