= Download B.A.T.M.A.N. =
We are currently working on different branches. To get the details about the differences of these branches, see our [wiki:BranchesExplained Branches Explained page].

We are offering sources and precompiled binaries and packages for different platforms and for the stable and the development branch. However, please use the stable branch for your public infrastructure unless you know exactly what you are doing and are prepared for the big unknown.

=== B.A.T.M.A.N. web interface package ===

The B.A.T.M.A.N. web interface package for the freifunk firmware as well as the latest stable batmand packages can be found in the ipkg repository at [http://freifunk.schmudde.com/ipkg/ freifunk.schmudde.com]. Therefore, if you are using freifunk firmware, the most convenient way to get B.A.T.M.A.N. running is probably to add the line
''src lui http://freifunk.schmudde.com/ipkg/'' to the file ''/etc/ipkg.conf'' on your WRT and then do an ''ipkg update'' and ''ipkg install batmand'' and ''ipkg install freifunk-batman'' (if not already installed you may also need the packages kmod-tun and libpthread ). Afterwards you can use the web interface to configure and observe batman.
